# 🏠 House Price Prediction

This project predicts house prices based on:

* Total Square Feet
* Number of Bathrooms
* Number of Balconies
* Number of BHK

It is built using **Machine Learning + Flask** and deployed online.

## 🚀 Live Demo

🔗 https://house-price-prediction-kooz.onrender.com
